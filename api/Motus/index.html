Motus

A lightweight animation handler wrapper for Roblox animator objects Provides simple methods for loading, playing, stopping, and managing animations. Handles caching and cleanup automatically.

Types​

AnimationInput​

</>
type AnimationInput = Animation | string | number

Represents an input that can be used to identify an animation.
Can be an Animation instance, an asset ID string ("rbxassetid://1234") or a numeric asset ID.
